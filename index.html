The point of this technical document is to educate the reader about some of the basics to using git and github. This is, by no-means a full documentation of git, but merely touches on the very basic functions.

What Is Git?

Git is one of the most widely used version control (VCS) systems in the world today. Git is an actively-maintained open source project developed by the creator of the Linux operating system, Linus Torvalds. Git is considered a distributed version control system (DVCS), meaning every developer on a project has a working copy of code, which is also a repository that can contain all changes made to a project by individual team members.
